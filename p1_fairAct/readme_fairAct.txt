This is project aims to solve real life problem of complaint management system.

This has 2 roles:
	1. Admin
		- Admin can see and update the status of the registered complaints (priority & category wise).
	2. Users
		- Users can see the status of the registered complaints, file new complaint or add support on existing problem.

This is very simple yet effective complaint management system.

Tech Stack:
	Front-End: HTML(4,5), CSS, Bootstrap(5), JavaScript(ES2025)
	Backend: PhP(8.4), JavaScript(ES2025)
	Data-base: MySQL (8.0)
	Hosting: Local Deployed on Xamp Aache Server.

This helps larger scale institutes to manage complaints effectively and transparently.
